# 🤖 Spring AI + Next.js Multi Chatbot Platform

A modern full-stack AI playground built using:

- ☕ Spring Boot 4
- 🧠 Spring AI
- ⚡ Next.js 16
- 🎨 Tailwind + DaisyUI
- 🗄️ Drizzle ORM
- 🤖 OpenAI + Ollama
- 🛠️ AI Tools
- 🔌 MCP (Model Context Protocol)
- 📚 RAG (Retrieval Augmented Generation)

This project is a collection of real-world AI chatbot implementations and experiments using modern Java + JavaScript technologies.

---

# ✨ Features

## 🤖 Multiple Chatbots

This project contains multiple chatbot implementations including:

- General AI assistant
- Persona based chatbots
- Custom role-based assistants
- Domain specific assistants
- Tool enabled chatbots
- Experimental AI agents

---

## 🎭 Persona Chatbots

Different AI personalities can be configured such as:

- Mentor
- Coding Assistant
- Stoic Philosopher
- Productivity Coach
- Interview Assistant
- Friendly AI Companion
- Technical Expert

Each chatbot can have:

- Custom system prompts
- Memory/context handling
- Different tools
- Different AI models
- Specialized behaviors

---

The backend demonstrates multiple Spring AI capabilities:

- Prompt Engineering

- Chat Memory

- RAG pipelines
